# Vedika-simple-Home-page-sample
Packlite's Simple Home page using Dart .
The Travel Checklist App interface created via the Flutter Code above uses MaterialApp as the foundation for the app architecture and a StatefulWidget for its stateful (dynamic) navigation.
The app has a BottomNavigationBar for three (3) main screens i.e. Home, Checklist and Dashboard. Selecting each tab in the BottomNavigationBar will cause the app to change its content displayed by the agreement of using Flutter's AnimationController to animate the icon when switching between tabs. 
The app uses the Scaffold class to build the AppBar and Drawer menu, as well as to define the page body layout.
